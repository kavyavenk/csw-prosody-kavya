# Data Files Note

Due to GitHub's file size limits, large data files are not included in this repository.

## Large Files Excluded

The following directories contain large files that should be obtained separately:

- `data/SEAME/interview_aligned/` - SEAME aligned audio files and TextGrids
- `data/SEAME/interview_mfa_files/` - MFA alignment files
- `data/*/wav16k/` - 16kHz audio files (will be generated)
- `data/*/wav16k_clips/` - Audio clips (will be generated)
- `*.zip` files - Original corpus archives

## How to Get the Data

1. **SEAME Corpus**: Download from the official SEAME repository
2. **MASAC Corpus**: Contact the corpus maintainers for access
3. **Generated files**: Run the scripts to generate audio clips and features

## Repository Contents

This repository contains:
- ✅ All Python scripts for processing
- ✅ Documentation (METHODOLOGY.md, etc.)
- ✅ Configuration files
- ✅ Small data files (CSVs, JSONs)
- ✅ Reports and summaries

The repository is functional for code and documentation, but requires the actual audio data to run the full pipeline.
